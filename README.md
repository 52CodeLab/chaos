### 🏷️ 版本号说明

本项目遵循[语义化版本 2.0.0](https://semver.org/lang/zh-CN/)

- `release` 版本号格式为 `x.x.x`
- `snapshots` 版本号格式为 `x.x.x-SNAPSHOT`

> `snapshots` 版本 `push` 后会自动发布，及时响应修复最新的 `bug` 和需求。
