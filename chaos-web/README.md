# Chaos 核心包

## 功能

1. Cglib Bean copy 增强，支持链式 bean、Map、优化性能和支持类型转换。
2. 基于hutool扩展的工具包
3. 基于SpringWeb扩展的工具包

## 添加依赖

### maven

```xml

<dependency>
    <groupId>club.gclmit</groupId>
    <artifactId>chaos-core</artifactId>
    <version>${version}</version>
</dependency>
```

### gradle

```groovy
compile("club.gclmit:chaos-core:${version}")
```
